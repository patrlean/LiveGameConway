# LiveGameConway

## Please first Compile the CMakeList under the SFML folder.
1. If no build folder under SFML folder, create a 'build' folder in the SFML folder
```
cd SFML
mkdir build
```
2. use cmake to compile the SFML.
```cmake
cmake ..
cmake --build .
```

1. create a 'build' folder in the LAB2 folder
2. use cmake to complie the SFML
